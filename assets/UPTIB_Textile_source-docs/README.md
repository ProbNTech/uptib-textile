# UPTIB Textile — Source Documents

These are the source reference documents for the **UPTIB Textile** website build (see `UPTIB_Textile_Website_BUILD_PROMPT.md`). They contain all the content, market data, strategy and supplier information the site is built from.

**Note:** The originals were supplied as `.docx` but were actually plain‑text / Markdown content (not real Word files). They have been delivered here as clean **Markdown (`.md`)** — ideal for a VS Code + Claude workflow (Claude reads them natively, they version‑control cleanly, and the content is unchanged apart from tidying minor formatting artifacts).

**Suggested location in your repo:** `/docs/source/` (or `/content/source/`).

## Brand note

The documents use both **UPTIB** and **UKPTIB**. Use **UPTIB** consistently across the website (this matches the live brand and the build prompt).
